AWS Polly Text-to-Speech
 
 This is a simple web application built using Node.js and AWS Polly. It converts text into natural-sounding speech using AWS cloud services.

📌 What this project does
- Convert text into speech
- Generate audio using AWS Polly
- Play the generated audio in the browser
- Simple frontend and backend integration

🛠️ Technologies Used
- Node.js
- Express.js
- AWS Polly
- HTML
- JavaScript
