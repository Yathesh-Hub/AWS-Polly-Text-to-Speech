# AWS Polly Text-to-Speech

This is a simple web application that converts text into speech using AWS Polly.

## Features
- Convert text to speech
- Simple frontend interface
- Node.js backend API
- Uses AWS Polly for voice generation

## Technologies Used
- Node.js
- Express.js
- AWS Polly
- HTML
- JavaScript

## Project Structure
AWS-Polly-Text-to-Speech
│
├── backend
│   └── mainserver.js
│
├── frontend
│   └── index.html
│
├── package.json
└── README.md

## Installation

1. Clone the repository

git clone <repository-link>

2. Go to the project folder

cd AWS-Polly-Text-to-Speech

3. Install dependencies

npm install

4. Run the server

node backend/mainserver.js

Server will run on:
http://localhost:3000
